Заголовок с названием проекта.
'Развитие игровой индустрии'

Цель проекта: Провести обзор игровых платформ, изучить объёмы продаж игр разных жанров и 
региональные предпочтения игроков. 
Изучить развитие игровой индустрии с 2000 по 2013 год.

Для выполнения проекта была использована таблица с информацией о продажах игр разных жанров и платформ, 
а также пользовательские и экспертные оценки игр, которая содержит 16956 строк
максимальное значение пропущенных данных в столбце amenities used — 54%.

для выполнения проекта я использовала библиотеку:
import pandas as pd 

Для оптимизации работы с данными в датафрейме были проведена предобработка данных: 
-удалены 2 строки с пропусками,
- в 275 строках пропуски дополнили средним значением,
- провели нормализация данных с текстовыми значениями,
- удалили 241 строку с дубликатами
Сохранен новый срез данных периода с 2000 по 2013 год включительно в отдельном датафрейме.

Вывод:

- По оценкам пользователей и критиков данные сгруппированы по категориям - высокая оценка, 
средняя оценка, низкая оценка.
- По каждой категории посчитано количество проданных игр.
- Выделены топ-7 платформ по количеству игр, выпущенных за весь актуальный период.
