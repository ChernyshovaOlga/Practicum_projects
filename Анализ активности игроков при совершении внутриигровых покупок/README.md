#проект
'Анализ активности игроков при совершении внутриигровых покупок'
индивидуальный проект о изучении развития игровой индустрии

Цель проекта: оценить активность игроков при совершении внутриигровых покупок. 

Вертикаль: Игровая индустрия
Решаемая проблема: привлечь платящих игроков и продвигать покупку эпических предметов с помощью рекламы

Что сделано:
- доля игроков покупает внутриигровую валюту «райские лепестки» за реальные деньги 
- зависимость доли платящих игроков от расы персонажа;
- детально изучено, как происходит покупка эпических предметов внутри игры:
1. Сколько внутриигровых покупок и что можно сказать об их стоимости (минимум и максимум, 
есть ли различие между средним значением и медианой, какой разброс данных
2. аномальные покупки по стоимости
Результат:
Данный отчет помогает оценить вовлеченность  аудитории.  Дает представление о поведении пользователей, их предпочтениях и готовности тратить деньги в игре.
Наблюдается зависимость между покупкой валюты «райские лепестки», сегментом игрока и характеристикой персонажа.
Предмет  Book of Legends востребован среди покупателей, его покупает практически каждый покупатель. Он может быть триггером для того, чтобы игрок перешел в покупатели внутри игры.
