#проект
'Исследовательский анализ концепции нового заведения общественного питания'

индивидуальный проект по исследованию рынка г. Москвы
Вертикаль: Бизнес-проект
Решаемая проблема: выбор подходящего инвесторам места нового заведения общественного питания

Что сделано:
- количество объектов общественного питания по каждой категории
- распределение количества заведений по административным районам Москвы, 
также отдельно распределение заведений каждой категории в Центральном административном округе Москвы.
- Изучили соотношение сетевых и несетевых заведений в целом по всем данным и в разрезе категорий заведения.
- Привели для каждой категории заведений наиболее типичное для него количество посадочных мест.
- Визуализировали распределение средних рейтингов по категориям заведений
- с какими данными показывают самую сильную корреляцию рейтинги заведений
- топ-15 популярных сетей в Москве
- средний чек заведения (столбец middle_avg_bill) в зависимости от района Москвы.

Результат:
На основе анализа следует обратить внимание на сетевые заведения заведения 
с большим рейтингом и меньшие по количеству, таких как:
    'теремок',
    'кулинарная_лавка_братьев_караваевых',
    'кофемания' .
Наибольший Средний чек в Центральном административном округе: 1191.06 руб.
