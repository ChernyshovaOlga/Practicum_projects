#проект
'Исследовательский анализ данных в Python и проверка гипотез.Знакомство с данными'

индивидуальный проект по выявлению инсайтов пользовательских предпочтений
Вертикаль: Онлайн сервис поиска мероприятий и продажи билетов
Решаемая проблема: прогнозирование выручки
Что сделано:
- Исследовательский анализ данных
- Анализ распределения заказов по сегментам и их сезонные изменения
- Проверена гипотеза о разнице в поведении пользователей с мобильных и стационарных устройств
Результат:
- В какое время года отмечается устойчивый рост количества заказов по различным сегментам
- Ключевые регионы, которые вносят наибольший вклад в выручку
Стек: SQL, Yandex DataLens, Python (pandas, matplotlib) , статистика

проверена гипотеза о разнице в поведении пользователей с мобильных и стационарных устройств:
- Среднее количество заказов на одного пользователя для мобильных устройств (2.87) выше, 
чем для стационарных устройств (1.98). Однако полученное p-значение (0.602) больше 0.05, 
что указывает на то, что эти различия не являются статистически значимыми. 
Это означает, что на основе имеющихся данных мы не можем с уверенностью утверждать, 
что разница в среднем количестве заказов между пользователями мобильных и стационарных 
устройств действительно существует.
- Среднее время между заказами у пользователей мобильных приложений выше по сравнению 
с пользователями стационарных устройств
