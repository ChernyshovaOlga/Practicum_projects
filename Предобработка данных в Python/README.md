#проект
'Развитие игровой индустрии'
индивидуальный проект о изучении развития игровой индустрии

Цель проекта: Провести обзор игровых платформ, изучить объёмы продаж игр разных жанров и 
региональные предпочтения игроков. 

Вертикаль: Игровая индустрия
Решаемая проблема: привлечь новую аудиторию

Что сделано:
- Загрузка и знакомство с данными
- Проверка ошибок в данных и их предобработка
- Преобразование типов данных
- Количество пропусков в каждом столбце в абсолютных и относительных значениях.
- Явные и неявные дубликаты в данных
- Нормализация данных с текстовыми значениями
- Фильтрация данных
- Категоризация данных

Результат:
- По оценкам пользователей и критиков данные сгруппированы по категориям - высокая оценка, 
средняя оценка, низкая оценка.
- По каждой категории посчитано количество проданных игр.
- Выделены топ-7 платформ по количеству игр, выпущенных за весь актуальный период.
Стек: Python (pandas)
